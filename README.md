## How to

### Install packages

```
$ npm install
```

### Update packages

```
$ npx npm-check-updates -u
$ npm install
```

### Build

```
$ npm run build
```

### Run (dev)

```
$ npm run dev
```

### Run (prod)

```
$ npm run start
```

### Report vulnerabilities

```
$ npm audit
```
